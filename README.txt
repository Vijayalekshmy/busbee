INSTALLATION
Run "Install PHSP_23.2-en_US-macuniversal" and follow the standard installation procedure.
When reporting damage, you must disable Gatekeeper, or make Decarantine.
https://appstorrent.ru/65-gatekeeper.html

sudo xattr -r -d com.apple.quarantine 
put a space and drag the corrupted program or image (.dmg) into the Terminal window

sudo spctl --master-disable
To enable Gatekeeper, replace "disable" with "enable".  

ACTIVATION
Move the "Adobe Photoshop 2022" file to the "macOS" folder and confirm the replacement.
Path: /Applications/Adobe Photoshop 2022/Adobe Photoshop 2022.app/Contents/MacOS

PLUGINS:
Camera Raw is installed for free from the official website.
https://helpx.adobe.com/camera-raw/kb/camera-raw-plug-in-installer.html
